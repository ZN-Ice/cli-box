# Phase 4: 高级特性

> 目标：完善沙箱能力，支持多窗口、录制回放、测试场景框架

## 任务清单

| 任务 ID | 描述 | 层 |
|---------|------|----|
| P4-01 | 多窗口支持：沙箱内管理多个子窗口（CLI + App 并存） | Rust |
| P4-02 | 录制功能：记录所有输入操作和截图，输出 JSONL | Rust |
| P4-03 | 回放功能：从 JSONL 文件回放操作序列 | Rust |
| P4-04 | 测试场景框架：YAML/JSON 定义测试步骤，自动执行 | Rust |
| P4-05 | 截图 diff：对比两张截图的差异（用于回归测试） | Rust |
| P4-06 | 集成测试报告：生成测试结果报告 | Rust |
| P4-07 | 发布：brew tap 或独立二进制分发 | Rust |

## 验收标准

- 沙箱内能同时运行 CLI 和 macOS App
- 录制能捕获所有操作和截图
- 回放能精确重现操作序列
- 测试场景能自动执行并报告结果
- 截图 diff 能检测 UI 变化
- 有完整的集成测试报告
